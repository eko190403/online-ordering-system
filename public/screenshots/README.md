# Screenshots

Folder ini berisi screenshot untuk dokumentasi README.md

## Cara Menambahkan Screenshot:

1. Ambil screenshot dari aplikasi (gunakan Windows Snipping Tool atau Print Screen)
2. Simpan dengan nama:
   - `qr-menu.png` - Tampilan menu customer
   - `cart-checkout.png` - Keranjang & checkout
   - `order-tracking.png` - Halaman tracking pesanan
   - `dashboard.png` - Dashboard admin
   - `order-management.png` - Kelola pesanan
   - `menu-management.png` - Kelola menu

3. Copy file ke folder ini
4. Screenshot akan otomatis muncul di README.md

## Ukuran Rekomendasi:
- Width: 1200px - 1920px
- Format: PNG (lebih tajam) atau JPG (lebih kecil)
- Compress jika file > 500KB
